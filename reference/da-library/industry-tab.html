ROW MEANINGS (detected by content, so order is flexible):
 • Tab label + background — a row of [label] | [background images]. The background cell holds up to THREE images: the desktop front layer, the desktop back layer (front is overlaid on back), and a single mobile image with the numbers baked in. Set each image's alt text to end in front, back, or mobile (filenames are hashed on upload, so alt is the reliable signal); if untagged, the block uses the order front, back, mobile. One image also works (used everywhere).
 • Portfolio CTA — a row containing a single link (the "Browse Complete … Portfolio" button).
 • Hotspot — a row of [label] | [x%, y%] | [popup image] | [popup description] | [popup link]. The x%, y% is where the toggle sits on the desktop image (percent from left, from top). Add one hotspot row per toggle. On mobile the toggles come from the baked-in image + the accordion, so positions are desktop-only. Replace the placeholder images with your own uploads.
